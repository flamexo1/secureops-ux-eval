# AI Tools Used

**Figma MCP (Claude + Model Context Protocol)**
Used to programmatically generate the wireframe screens in Figma — frames, layout, text, and color — directly through Claude, which sped up the execution significantly.

All design decisions, information architecture, annotations, and content were done by me. The MCP handled the repetitive Figma build work so I could focus on the thinking.

> AI tools help with speed. Design judgment is still mine.
